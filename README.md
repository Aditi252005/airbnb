☕ BrewScore – Cafe Rating & Review Platform

BrewScore is a full-stack web application that allows users to discover cafes, share reviews, and rate them based on multiple aspects such as food quality, ambience, price, and staff service. The platform helps users explore cafes and make informed dining decisions based on community feedback.

🚀 Features

🔐 User Authentication – Secure signup and login system using Passport.js
⭐ Multi-Criteria Ratings – Users can rate cafes on food, ambience, price, and staff
📝 Review System – Users can write and manage detailed reviews
📸 Image Uploads – Cafe images are uploaded and stored using Cloudinary
🔍 Search Functionality – Search cafes by name for easy discovery
📊 Dynamic Rating Aggregation – Average ratings and review counts are automatically updated
🗑 Authorization – Users can edit or delete only their own listings and reviews

🛠 Tech Stack
Frontend- HTML, CSS, Bootstrap, EJS
Backend- Node.js, Express.js
Database- MongoDB, Mongoose
Other Tools- Passport.js, Cloudinary, Render
